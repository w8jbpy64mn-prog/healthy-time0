# Healthy Time | وقت الصحة

هذا المستودع يحتوي على **الأساس المعماري والتشغيلي** لتنفيذ نظام Healthy Time بشكل فعلي.

## المحتويات

- `docs/IMPLEMENTATION_PLAN_AR.md`: خطة تنفيذ شاملة تغطي كل المتطلبات التشغيلية.
- `docs/DATABASE_SCHEMA.sql`: مخطط قاعدة بيانات PostgreSQL مبدئي جاهز للتطوير.
- `api/openapi.yaml`: عقد API أولي لواجهات الإدارة والعميل.

## الهدف الحالي

تحويل البريف التشغيلي الكامل إلى:
1. تصميم نظام قابل للتوسع
2. قواعد بيانات واضحة
3. واجهات API قابلة للتنفيذ

## الخطوة التالية المقترحة

بدء المرحلة الأولى (Core) عبر إنشاء مشروع Next.js + PostgreSQL + Prisma وتفعيل:
- تسجيل دخول الإدارة
- إدارة العملاء والباقات والمنتجات
- إنشاء رابط العميل
- عدادات الاشتراك الأساسية
